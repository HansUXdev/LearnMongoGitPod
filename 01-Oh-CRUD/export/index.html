What is JSON?

  • Stands for, JavaScript Object Notation
  • Curly braces, {} hold objects
  • Square brackets, [] hold arrays
  • Data is in name/value pairs
  • keys must be strings, written with double quotes:
    { "name":"John" }
  • Data is separated by commas
    { "name":"John" },
{ "name":"Jane" }
